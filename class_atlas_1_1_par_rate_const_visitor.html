Detailed Description

template<typename adouble>
class Atlas::ParRateConstVisitor< adouble >

A class for calculating the par rate of an instrument. It is a const visitor, so it does not modify the instrument. It calculates the needed spread, rate or other value that makes the NPV of the instrument equal to zero.

Template Parameters
adoubleThe number type used for the calculations. Can be double or an AAD type.
