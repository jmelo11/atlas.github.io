Detailed Description

template<typename adouble>
class Atlas::SensibilityConstVisitor< adouble >

Visitor in charge of calculating some sensibilities of a product.

For fixed income, the sensibility is calculated w.r.t the coupon rate (fixed rate instruments) or the coupon spread (floating rate instruments).
