Detailed Description

template<typename adouble>
class Atlas::FloatingRateCoupon< adouble >

A class representing a floating rate coupon.

A floating rate coupon has a start date, an end date, a notional amount, a spread, and a rate index

Template Parameters
adoubleThe type of the floating point number used in the coupon
