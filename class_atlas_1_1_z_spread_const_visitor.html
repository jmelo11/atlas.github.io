Detailed Description

template<typename adouble>
class Atlas::ZSpreadConstVisitor< adouble >

A visitor for calculating the z-spread of an instrument. Only fix income instruments are supported.

Template Parameters
adoubleThe number type used for the calculations. Can be double or an AAD type.
