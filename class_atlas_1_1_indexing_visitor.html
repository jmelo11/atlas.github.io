Detailed Description

template<typename adouble>
class Atlas::IndexingVisitor< adouble >

A class for indexing the cashflows of an instrument. Indexing is needed to compute all market variables ralated to the instrument's cashflows.

It's a Visitor, so it updates the state of the given instrument with the relevant indices.

Template Parameters
adoubleThe number type used for the calculations. Can be double or an AAD type.
