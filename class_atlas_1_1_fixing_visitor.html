Detailed Description

template<typename adouble>
class Atlas::FixingVisitor< adouble >

A visitor that fixes the coupons of an instrument with floating rate legs. After the fixing, the instrument can be priced.

sets the variable isFixingSet_ to true after fixing.

Template Parameters
adoubleThe number type used for the calculations. Can be double or an AAD type.
